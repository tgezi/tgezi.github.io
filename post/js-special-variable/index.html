Js 神奇的变量

2021-07-11

假设有个变量a，然后看以下判断:

a === 1 // true
a == 2 // true
a == 3 && a == 4 && a == 5 // true
a // 6
a == 6 // false

这个变量的原理就是不断自增，没当获取它的时候就自增1，要做到这个效果可以用数据劫持来做，就是Object.defineProperty

var count = 1; // 用来自增的变量
Object.defineProperty(window,"a",{ // 第一个参数是要劫持的对象，第二个是属性名(变量名)，第三个是详细的配置
    get(){  // 当获取值时的函数
        value = count++; // 属性的值等于count自增后的结果
        return value; // 返回结果
    },
    set(val){ // val是设置变量时，等号右边的值
        value = val; // 将属性的值设为参数
        return value; // 返回结果
    }
});

后面就可以实现这个"万能变量"了
